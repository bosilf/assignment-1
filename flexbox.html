FLEX

flex-direction: row; is actually the default flex-direction, so you don't need to specify it.
To spice things up a little you can justify-content. It decides how items align along the main axis of the current line of the flex container.
Here are some different examples of what a flex-direction: row; can look like:

justify-content: flex-end;

Since flex-direction: row; is the default flex-direction, you don't need to specify it.
So, if you want the content in a different direction, one way you can do it is flex-direction: row-reverse;

Besides the directions row and row-reverse there is also flex-direction: column; and flex-direction: column-reverse;
With these you can use the same justify-contents as the rows use but in columns.
